## BuildaLab — The Full Picture

One engineer. Full production SaaS. Real revenue.

| Layer | What I Built |
|---|---|
| **Backend** | NestJS API with Fastify, TypeORM, BullMQ job queues, Cognito auth, role-based access control |
| **Frontend** | Next.js 15 + React 19 with Zustand, TanStack Query, Zod validation, and shadcn/ui |
| **AI Pipelines** | LLM integrations (OpenAI, Gemini) for structural optimization and intelligent analysis |
| **Infrastructure** | ECS Fargate, RDS PostgreSQL, ElastiCache Redis, S3, SES, CloudWatch — all on AWS |
| **Architecture** | Turborepo monorepo, Yarn Workspaces, shared type packages, strict TypeScript |
| **DevOps** | GitHub Actions CI/CD, Docker multi-stage builds, automated testing & deployment |
