## What I've Built in Production

> Single-handedly designed, built, and maintain a full SaaS platform handling real workloads.

### Backend & API Design
- REST API with **NestJS** + **Fastify**, modular architecture with guards, interceptors, and validation pipes
- **TypeORM** with PostgreSQL — migrations, soft deletes, query optimization, transactional workflows
- Role-based access control with **AWS Cognito** JWT authentication and custom guard chains
- Async job processing with **BullMQ** + **Redis** — concurrency control, retry strategies, resource locking
- Structured logging, correlation IDs, audit trails, and standardized error handling across all services

### Full-Stack
- **Next.js 15** + **React 19** frontend with server components and streaming
- State management with **Zustand** (client) and **TanStack Query** (server state, caching, optimistic updates)
- Form handling with **React Hook Form** + **Zod** validation
- Token management with auto-refresh, silent renewal, and session handling

### Infrastructure & DevOps
- **AWS**: ECS Fargate, RDS PostgreSQL, ElastiCache Redis, S3, SES, VPC, CloudWatch
- **CI/CD**: GitHub Actions pipelines — lint, test, build, deploy
- **Monorepo**: Turborepo + Yarn Workspaces with shared type-safe packages
- Docker multi-stage builds, environment-based configuration, health checks

### AI Integration
- LLM pipelines using **OpenAI** and **Google Gemini** APIs for structural analysis optimization
- Queue-based orchestration for long-running AI workloads with status tracking and result processing

## Engineering Practices
- Comprehensive **unit and integration testing** with Jest — factories, mocks, database cleanup strategies
- Strict **TypeScript** across the entire stack — no `any`, shared DTOs between backend and frontend
- API documentation with **Swagger/OpenAPI**, auto-generated from decorators
- Rate limiting, input validation, CORS, security headers — OWASP-aware from day one
